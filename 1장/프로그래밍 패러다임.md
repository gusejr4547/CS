## 1. 선언형과 함수형 프로그래밍
선언형 프로그래밍 : '무엇을' 풀어내는가에 집중하는 패러다임.

"프로그램은 함수로 이루어진 것이다."

함수형 프로그래밍은 선언형 패러다임의 일종

### 순수 함수
출력이 입력에만 의존하는 것
```
void pure(a,b){
    return a + b;
}
```
들어오는 매개변수 a,b에만 영향을 받는다. 다른 전역변수 c 등이 출력에 영향을 주면 순수 함수가 아니다.
